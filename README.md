# CTF Writeups - Software Security

This repository contains writeups and scripts for CTF (Capture The Flag) challenges developed for the **Software Security** course.

### Buffer Overflow
Exploitation of buffer overflow vulnerabilities, including:
- Simple overflow
- Return address manipulation
- Calling specific functions

### Format String
Exploitation of format string vulnerabilities in C, covering:
- Local memory reading
- Writing to arbitrary addresses
- Overwriting specific values
- Execution flow manipulation

### Race Conditions
Challenges involving race conditions and timing exploitation:
- File system race conditions
- Serialization exploitation
- Process synchronization

### Remote Server
Interaction and exploitation of remote servers:
- Using libraries like `requests` and `pwntools`
- Network service exploitation
- Authentication bypass

### SQL Injection (SQLi)
Exploitation of SQL injection vulnerabilities:
- Classic SQL injection
- Blind SQL injection
- Sensitive data extraction

### Cross-Site Scripting (XSS)
Filter bypass and XSS exploitation:
- WAF (Web Application Firewall) bypass
- Cookie theft
- Input sanitization bypass
